# 경비 처리 및 출장 규정

## 1. 출장비 규정

### 1.1 국내 출장
- 일비: 일 30,000원 (당일 출장 20,000원)
- 숙박비: 실비 정산, 1박 최대 100,000원 (서울/수도권), 80,000원 (기타 지역)
- 교통비: 실비 정산 (영수증 첨부 필수)
  - KTX/항공: 이코노미 기준 실비
  - 자가용 이용: km당 200원
- 식비: 조식 8,000원, 중식 10,000원, 석식 12,000원 (영수증 불필요)

### 1.2 해외 출장
- 일비: 국가별 상이 (인사부 출장 규정 참조)
  - 미주/유럽: USD 60 상당
  - 동남아시아: USD 40 상당
  - 일본/중국: USD 50 상당
- 숙박: 실비 (비즈니스급 기준)
- 항공: 이코노미 (10시간 이상 비즈니스 가능, 팀장 이상 사전 승인 필요)
- 여행자 보험: 회사 단체 보험 자동 적용

### 1.3 출장 신청 절차
1. 사내 인사포털(hr.company.com) → 출장 신청 메뉴
2. 출장 목적, 기간, 방문처, 예상 경비 입력
3. 팀장 승인 (해외 출장은 본부장 승인)
4. 출장 전 개인 카드 또는 법인카드 사용

### 1.4 출장비 정산 절차
1. 출장 복귀 후 5영업일 이내 정산
2. 인사포털 → 출장비 정산 신청
3. 영수증 원본 스캔 첨부 (5만원 이상 건별)
4. 담당자 확인 후 익월 급여일에 지급

## 2. 업무 경비 처리

### 2.1 법인카드 사용 기준
- 사용 가능: 업무용 물품 구매, 거래처 식대, 출장 교통비
- 사용 불가: 개인 용도, 가족 동반 식사, 주류 단독 구매

### 2.2 개인카드 실비 정산
- 법인카드 사용 불가 상황 시 개인카드 사용 가능
- 사후 정산 신청: 인사포털 → 경비정산 신청
- 정산 기한: 비용 발생일로부터 30일 이내

### 2.3 접대비
- 거래처 식사/선물: 1인당 3만원 이내 (단, 1회 10만원 이상 시 본부장 사전 승인)
- 명절 선물: 1인당 5만원 이내

## 3. 복리후생 지원금

### 3.1 경조사비
- 본인/배우자 결혼: 300,000원
- 본인 및 배우자 부모 사망: 300,000원
- 본인 사망: 500,000원
- 자녀 결혼: 100,000원

### 3.2 자기 계발비
- 연간 300,000원 지원 (1월 1일 기준 재직자)
- 사용 항목: 도서 구매, 자격증 취득, 어학 시험
- 사용 방법: 영수증 제출 → 인사팀 확인 → 당월 급여에 추가 지급
