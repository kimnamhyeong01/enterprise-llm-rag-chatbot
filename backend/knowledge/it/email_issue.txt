# IT 시스템 장애 및 헬프데스크 가이드

## 1. 이메일 관련 문제

### 1.1 이메일 용량 초과 해결 방법
증상: "메일함 용량이 초과되었습니다" 알림

해결 단계:
1. 받은 편지함 정리 (대용량 첨부 파일 메일 삭제)
   - Outlook: 정렬 기준 → 크기 순 정렬 → 불필요 메일 삭제
2. 삭제된 항목 폴더 비우기
   - Outlook: 삭제된 항목 우클릭 → "삭제된 항목 폴더 비우기"
3. 보낸 편지함 정리 (대용량 첨부 파일 보낸 메일 삭제)
4. 정크 메일 폴더 비우기

추가 조치:
- 대용량 파일(10MB 이상) 공유 시 이메일 첨부 대신 사내 파일 서버 링크 사용
- 메일함 용량 증설 필요 시 IT헬프데스크 요청 (내선: 5000)
- 현재 메일함 크기 확인: Outlook → 파일 → 계정 설정 → 데이터 파일

### 1.2 이메일 수신/발신 오류
- 아웃룩 연결 오류: 오른쪽 하단 아웃룩 상태 표시 확인
  - "연결 끊김": 인터넷 연결 또는 VPN 상태 확인
  - "오프라인으로 작업 중": Outlook → 보내기/받기 탭 → 오프라인 작업 해제
- 외부 발송 오류: 수신자 이메일 주소 확인, 스팸 필터링 여부 확인

## 2. SAP 오류

### 2.1 SAP 로그인 오류
**오류: "계정이 잠겼습니다"**
- 비밀번호 5회 오류 시 계정 잠금
- 해결: IT헬프데스크(내선: 5000) 또는 SAP 담당자(내선: 5100)에게 잠금 해제 요청

**오류: "시스템에 접속할 수 없습니다"**
1. SAP GUI 재시작
2. 서버 접속 설정 확인: 서버 주소 sap.company.com, 시스템 번호 00
3. VPN 연결 상태 확인 (재택 시)
4. 문제 지속 시 SAP 담당자 연락

**오류: "권한 없음"**
- 특정 트랜잭션 접근 권한 필요
- 해결: 소속 부서 팀장 → IT 권한 신청서 제출 → IT팀 처리 (2~3 영업일)

### 2.2 SAP 응답 느림
1. 현재 네트워크 속도 확인 (speedtest.net)
2. SAP GUI 창 닫기 후 재접속
3. 시스템 피크 시간(08:30~09:30, 17:30~18:00) 혹은 월말 마감 시 지연 가능
4. 문제 지속 시 SAP 시스템팀 연락 (내선: 5100)

## 3. 기타 IT 문제

### 3.1 PC 부팅 불가
1. 전원 케이블 연결 상태 확인
2. 강제 재시작: 전원 버튼 5초 이상 누름 → 재부팅
3. 부팅 중 오류 메시지 사진 촬영 후 IT헬프데스크 접수

### 3.2 프린터 연결 오류
1. 프린터 전원 및 네트워크 케이블 확인
2. Windows: 설정 → 프린터 및 스캐너 → 프린터 제거 후 재추가
3. 사내 프린터 IP: 10.10.5.x 대역 (층별 상이, IT 포털 참조)
