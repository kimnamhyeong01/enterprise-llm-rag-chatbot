# 설비 운영 매뉴얼

## 1. 프레스 설비 운영 가이드

### 1.1 일상 점검 항목
- 작동 전 유압 오일 레벨 확인 (정상 범위: 80~100%)
- 안전 가드 체결 상태 확인
- 비상 정지 버튼 동작 확인
- 프레스 압력 게이지 확인 (정상: 150~200 bar)

### 1.2 A장비 (프레스 PA-100) 필터 교체 주기
- 유압 필터: 2,000시간 또는 6개월 주기
- 에어 필터: 500시간 또는 3개월 주기
- 오일 필터: 1,000시간 주기
- 교체 작업 전 반드시 설비 정지 후 잠금 장치 적용 (LOTO 절차 준수)

### 1.3 B장비 (컨베이어 CV-200) 필터 교체 주기
- 구동 모터 에어 필터: 250시간 또는 2개월 주기
- 제어판 필터: 1,000시간 또는 6개월 주기

## 2. 3번 라인 (Line-3) 운영 가이드

### 2.1 압력 이상 알람 조치 절차
알람 코드: PA-301 (압력 초과), PA-302 (압력 부족)

PA-301 (압력 초과) 조치:
1. 즉시 컨베이어 속도 50% 감속 (HMI 화면 → 속도 조절 탭)
2. 라인 헤드부 및 이송부 이물질 확인
3. 구동 모터 과부하 확인 (전류계 정상 범위: 15~20A)
4. 이상 없을 경우 15분 냉각 후 정상 속도로 재기동
5. 알람 재발 시 설비팀 즉시 신고 (내선: 3241)

PA-302 (압력 부족) 조치:
1. 에어 공급 라인 메인 밸브 개방 여부 확인
2. 에어 컴프레서 압력 게이지 확인 (정상: 6~8 bar)
3. 에어 호스 연결 상태 및 누기 여부 점검
4. 문제 지속 시 설비팀 연락 (내선: 3241)

### 2.2 라인 재기동 절차
1. 비상 정지 상황 해제 확인
2. HMI 화면에서 알람 리셋
3. 안전 가드 체결 상태 재확인
4. 서행 모드로 초기 가동 (정상 속도의 30%)
5. 이상 없을 시 단계적 속도 상향

## 3. 에러 코드 목록

| 에러코드 | 의미 | 조치 |
|---------|------|------|
| E-201 | 유압 압력 이상 | 유압 계통 점검, 오일 누출 확인 |
| E-202 | 온도 과열 (80°C 초과) | 냉각 시스템 점검, 작동 중지 후 냉각 |
| E-203 | 모터 과부하 | 부하 감소 후 재기동, 반복 시 모터 점검 |
| E-204 | 센서 통신 오류 | 센서 케이블 재결합 후 PLC 재부팅 |
| E-205 | 비상 정지 활성화 | E-Stop 해제 후 안전 확인 절차 수행 |
| E-206 | 오일 부족 (30% 미만) | 지정 오일 보충 후 재기동 |
| E-207 | 과전류 감지 | 전원 차단 후 전기 담당자 연락 |

### E-204 (센서 통신 오류) 상세 조치
E-204는 센서와 PLC 간 통신 오류를 의미합니다.

조치 순서:
1. 해당 센서 케이블 커넥터 탈거 후 재결합
2. 케이블 손상 또는 굴곡 심한 부위 확인
3. PLC 제어판 재부팅 (전원 OFF → 10초 대기 → ON)
4. 재기동 후 센서 신호 정상 여부 모니터링 (5분 관찰)
5. 오류 지속 시 유지보수팀 연락 (내선: 3300)
6. 센서 교체 필요 시 부품 청구 (자재창고 내선: 3150)
