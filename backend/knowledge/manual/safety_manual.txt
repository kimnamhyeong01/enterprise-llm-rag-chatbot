# 안전 규정 및 SOP (Standard Operating Procedure)

## 1. 기본 안전 수칙

### 1.1 개인 보호 장구 (PPE) 착용 기준
- 안전모: 공장 내 전 구역 필수
- 안전화: 공장 내 전 구역 필수
- 방진 마스크: 도장 구역, 분진 발생 구역
- 내화학 장갑: 화학물질 취급 구역
- 차광 보안경: 용접 구역 및 강한 빛 발생 구역

### 1.2 LOTO (잠금/표지) 절차
설비 정비, 청소, 조정 작업 전 반드시 수행합니다.

절차:
1. 설비 전원 OFF
2. 에너지 차단 장치에 개인 잠금장치 부착
3. 빨간 LOTO 태그 부착 (작업자 성명, 작업 내용, 날짜 기재)
4. 에너지 방출 확인 (잔류 에너지 제거)
5. 작업 완료 후 잠금장치 본인이 직접 해제

## 2. 화재 및 비상 대피 절차

### 2.1 화재 발생 시 행동 요령
1. 화재 경보 발령 (비상벨 누름)
2. 119 신고 (또는 내선 119)
3. 초기 소화기 사용 (30초 이내 진화 불가 시 즉시 대피)
4. 지정 대피로로 대피 (엘리베이터 사용 금지)
5. 집결 지점: 정문 앞 주차장 (A구역)

### 2.2 대피 집결 지점
- A라인~C라인: 정문 앞 주차장 A구역
- D라인~F라인: 후문 공터 B구역
- 사무동: 본관 앞 광장

## 3. 화학물질 취급 안전

### 3.1 절삭유 취급
- 피부 접촉 시: 즉시 흐르는 물로 15분 세척
- 눈 접촉 시: 세안 시설에서 15분 이상 세척 후 의무실 방문
- 누출 시: 흡수 키트 사용, 하수구 유입 방지

### 3.2 유해 화학물질 보관
- 반드시 지정 보관 장소 사용
- 혼재 보관 금지 (산/염기, 산화제/가연물)
- MSDS (물질 안전 보건 자료) 비치 확인

## 4. 중대 산업 재해 예방 수칙

- 2인 1조 작업 원칙: 위험 구역 작업 시 단독 작업 금지
- 안전 작업 허가서: 고소 작업, 밀폐 공간 작업 시 발급 필수
- 안전 점검: 매일 작업 전 5분 안전 점검 실시
- 위험 신고: 위험 요소 발견 시 즉시 안전팀 신고 (내선: 9119)
