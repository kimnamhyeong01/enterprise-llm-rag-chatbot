# 트러블슈팅 가이드

## 1. 자주 발생하는 설비 문제

### 1.1 컨베이어 벨트 이탈
증상: 벨트가 한쪽으로 쏠리거나 이탈

조치:
1. 즉시 설비 정지 (비상 정지 버튼)
2. 벨트 장력 확인 (정상 처짐량: 10~15mm)
3. 트래킹 롤러 위치 조정 (우측으로 쏠릴 경우 좌측 롤러 전진)
4. 재기동 후 5분 관찰
5. 재발 시 벨트 교체 필요 → 설비팀 요청

### 1.2 유압 실린더 동작 불량
증상: 실린더가 정상 속도보다 느리거나 멈춤

원인 및 조치:
- 오일 부족: 오일 탱크 레벨 확인 및 보충 (SAE 46 유압유)
- 필터 막힘: 유압 필터 교체 (2,000시간 주기 확인)
- 제어 밸브 이상: 밸브 청소 또는 교체
- 배관 누유: 피팅부 점검 및 조임

### 1.3 모터 과열 (E-202)
증상: 모터 표면 온도 80°C 초과, 알람 발생

조치:
1. 즉시 설비 정지
2. 20분 자연 냉각
3. 모터 주변 이물질 및 통기 막힘 확인
4. 냉각 팬 동작 여부 확인
5. 부하 과다 여부 검토 (생산량 조정 검토)
6. 재발 시 모터 교체 검토 → 설비팀 연락

### 1.4 PLC 통신 오류
증상: HMI 화면 "통신 오류" 또는 "연결 끊김" 표시

조치:
1. PLC 전원 상태 확인 (정상: 녹색 LED 점등)
2. 이더넷 케이블 연결 상태 확인
3. PLC 재부팅 (전원 OFF → 30초 대기 → ON)
4. HMI 소프트웨어 재시작
5. 문제 지속 시 IT/자동화팀 연락 (내선: 3500)

## 2. 품질 관련 문제

### 2.1 치수 불량 발생 시
1. 불량품 즉시 격리 (빨간 태그 부착, 불량품 박스 이동)
2. 최근 100개 제품 샘플링 검사
3. 치수 측정 후 편차 기록
4. 설비 원점 확인 및 재조정
5. 품질팀 통보 (내선: 4100)

### 2.2 표면 불량 (스크래치, 이물질) 발생 시
1. 불량품 격리
2. 원인 파악 (공구 마모, 이물질 혼입, 설비 이상)
3. 공구 교체 또는 공정 세척
4. 초도품 검사 후 양산 재개
