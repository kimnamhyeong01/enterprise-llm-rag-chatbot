# 공정 및 품질 가이드

## 1. 생산 공정 전체 흐름

### 1.1 주요 공정 순서
```
원자재 입고 → 자재 검사 → 창고 보관
    ↓
프레스 가공 (Line 1~3)
    ↓
표면 처리 (도금/도장)
    ↓
조립 공정
    ↓
품질 검사 (중간 검사 + 완성품 검사)
    ↓
포장 및 출하 검사
    ↓
출하
```

### 1.2 각 공정 상세

**원자재 입고 및 검사**
- 납품 업체로부터 원자재(코일, 판재) 입고
- 수입 검사: 치수, 표면 상태, 재질 성적서 확인
- 합격: 창고 입고 / 불합격: 반품 처리

**프레스 가공 공정**
- 코일 언코일러 → 피더 → 프레스 → 이송
- 1차 공정: 블랭킹 (외형 타발)
- 2차 공정: 드로잉 (성형)
- 3차 공정: 트리밍/피어싱 (불필요 부분 제거, 구멍 가공)

**표면 처리**
- 도금: 아연 도금 (방청 목적), 두께 8~12μm
- 도장: 에폭시 분체 도장 (색상: 고객사 지정)
- 처리 후 부착력 테스트 필수

**조립 공정**
- 반자동/자동 조립 라인 운영
- 토크 관리: 전동 드라이버 사용, 토크 기록지 작성
- 서브 조립 → 메인 조립 → 기능 검사

## 2. 품질 검사 기준

### 2.1 수입 검사 기준
- 검사 수준: AQL 2.5 적용 (KS Q ISO 2859-1)
- 치수 허용 오차: 도면 공차 기준 (일반 ±0.1mm)
- 표면 결함: 스크래치 0.5mm 이상, 찍힘 0.3mm 이상 불합격

### 2.2 공정 검사 기준 (초도품/중간 검사)
**초도품 검사**:
- 매 생산 시작 시 첫 5개 전수 검사
- 치수, 외관 모두 합격 시 양산 진행
- 불합격 시 원인 분석 후 재조정

**중간 검사 (공정 검사)**:
- 매 1시간마다 샘플 5개 검사
- 치수 측정 결과 공정 관리도(X-bar R 관리도)에 기록

### 2.3 완성품 검사 기준
| 항목 | 기준 | 방법 |
|------|------|------|
| 치수 | 도면 공차 이내 | 버니어 캘리퍼스, 3D 측정기 |
| 외관 | 스크래치, 이물질 없음 | 육안 검사 (조도 1000 Lux 이상) |
| 기능 | 고객 사양 기능 100% 작동 | 기능 시험 장치 |
| 도금 두께 | 8~12μm | 도금 두께 측정기 |

### 2.4 불량 처리 절차
1. 불량 발견 즉시 별도 구역 격리 (빨간 태그 부착)
2. 불량 내용 기록: 불량 유형, 수량, 발생 공정
3. 품질팀 통보 (내선: 4100)
4. 원인 분석 (4M: Man, Machine, Material, Method)
5. 시정 조치 실시 및 재발 방지 대책 수립
6. 불량품 처리: 재작업 가능 여부 판단 → 재작업 또는 폐기

## 3. 생산 KPI

- 가동률: 목표 95% 이상 (계획 대비 실제 가동 시간)
- 불량률: 목표 0.5% 이하 (전체 생산량 대비 불량 수량)
- OEE (종합 설비 효율): 목표 85% 이상
- 납기 준수율: 목표 98% 이상
