# 기능 테스트 시나리오

본 문서는 전사 RAG 챗봇 시스템의 기능 검증을 위한 테스트 시나리오를 정의한다.

---

## 테스트 환경 사전 확인

1. Docker Desktop 실행 확인
2. `docker-compose up --build` 실행
3. 백엔드 헬스 체크: `curl http://localhost:8000/health` → `{"status": "ok"}` 확인
4. 프론트엔드 접속: `http://localhost:8501`

---

## 시나리오 1: 현장 매뉴얼 챗봇 (도메인: manual)

### TC-1-1: 압력 이상 알람 조치

**입력 질문:**
```
3번 라인에서 압력 이상 알람 뜨면 어떻게 조치해야 해?
```

**기대 결과:**
- 라우팅 도메인: `🔧 현장 매뉴얼`
- PA-301 또는 PA-302 알람 코드 및 단계별 조치 방법 설명
- 참고 문서: `equipment_manual.txt`

---

### TC-1-2: 장비 필터 교체 주기

**입력 질문:**
```
A장비 필터 교체 주기는?
```

**기대 결과:**
- 라우팅 도메인: `🔧 현장 매뉴얼`
- 유압 필터(6개월/2,000시간), 에어 필터(3개월/500시간), 오일 필터(1,000시간) 안내
- 참고 문서: `equipment_manual.txt`

---

### TC-1-3: 에러 코드 의미

**입력 질문:**
```
에러 코드 E-204 의미가 뭐야?
```

**기대 결과:**
- 라우팅 도메인: `🔧 현장 매뉴얼`
- 센서 통신 오류 설명 및 6단계 조치 절차 안내
- 유지보수팀 내선 번호(3300) 포함

---

### TC-1-4: 이미지 업로드 (선택)

**입력 방법:**
1. 사이드바에서 현장 설비 사진 업로드
2. "이 장비 어떤 문제가 있어 보여?" 질문

**기대 결과:**
- 이미지 분석 결과 + 관련 매뉴얼 기반 답변

---

## 시나리오 2: HR 챗봇 (도메인: hr)

**입력 질문:**

### TC-2-2: 연차 이월 규칙

**입력 질문:**
```
연차 이월 규칙이 어떻게 돼?
```

**기대 결과:**

- 참고 문서: `vacation_policy.txt`

---

**입력 질문:**


**기대 결과:**
- 라우팅 도메인: `👤 사내 규정/HR`
- hr.company.com 인사포털 신청 → 3일 전 신청 → 팀장 승인 절차 안내
- 코어 타임 및 주 최대 2일 제한 정보

## 시나리오 3: IT 헬프데스크 (도메인: it)

### TC-3-1: VPN 접속 문제

**입력 질문:**
```
VPN 접속이 안돼
```

**기대 결과:**
- 라우팅 도메인: `💻 IT 헬프데스크`
- 클라이언트 미설치, 비밀번호 오류, MFA 문제, 방화벽 차단 등 원인별 해결 방법 안내
- 참고 문서: `vpn_troubleshooting.txt`

---

### TC-3-2: SAP 로그인 오류

**입력 질문:**
```
SAP 로그인이 안 돼. 계정이 잠겼다고 나와
```

**기대 결과:**
- 라우팅 도메인: `💻 IT 헬프데스크`
- 비밀번호 5회 오류로 계정 잠금 설명 + IT헬프데스크(내선 5000) 또는 SAP 담당자(내선 5100) 연락 안내

---

### TC-3-3: 이메일 용량 초과

**입력 질문:**
```
메일 용량 초과 해결 방법이 뭐야?
```

**기대 결과:**
- 라우팅 도메인: `💻 IT 헬프데스크`
- 받은 편지함 정리(크기 기준 정렬), 삭제된 항목 비우기, 보낸 편지함 정리 등 단계별 안내
- 참고 문서: `email_issue.txt`

## 시나리오 4: 영업/고객 대응 (도메인: sales)

### TC-4-1: 제품 비교

**입력 질문:**
```
A제품과 B제품 차이 설명해줘
```

**기대 결과:**
- 라우팅 도메인: `📊 영업/고객 대응`
- 압력(500ton vs 200ton), 정밀도(±0.01 vs ±0.05mm), 가격, 특징 비교표 제공
- 참고 문서: `product_info.txt`

---

### TC-4-2: 고객 질문 대응

**입력 질문:**
```
고객이 경쟁사 제품이랑 비교해달라고 하면 어떻게 답해?
```

**기대 결과:**
- 라우팅 도메인: `📊 영업/고객 대응`
- 경쟁사 A사/B사 대비 당사 강점(에너지 절감, IoT 기능, AS 속도) 안내
- TCO 비교 및 대응 전략 제시
- 참고 문서: `competitor_analysis.txt`

## 성능 기준

- 응답 시간: 10초 이내
- 도메인 라우팅 정확도: 90% 이상 (위 10개 케이스 기준)
- 소스 문서 반환: 모든 응답에 참고 문서 1개 이상 포함
